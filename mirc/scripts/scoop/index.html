Scoop Script 2004

O Scøøp Script é um script para acessar os canais de bate papo do IRC pelo Brasil, Portugal ou nas redes Internacionais.

Novidades:

  • mIRC 6.12 em português adicionado;
  • Mudança no visual do Script com novo theme padrão e novo ícone;
  • Sistema de Themes melhorado;
  • Novos utilitários e atualização dos já existentes:
    • Winamp Linker;
    • Sistema de torpedo para enviar mensagens para celulares;
    • Sistema de mudança das tonalidades das cores padrão do script;
    • Tradutor online com novo sistema;
    • Importador de Notify List (/inotify);
    • Editor de aliases (/ealias);
    • Editor de Risadas;
  • Carinhas coloridas com acessórios;
  • Sistema de notificações com balões;
  • Sistema de proteções e defesa melhorado e atualizado;
  • Novos temas no Gama Game e Ask game;
  • Adicionadas dezenas de pensamentos, frases de amor e piadas;
  • Lista de servidores ampliada e atualizada;
  • Correção de vários erros da versão anterior;
  • Dentre muitas outras mudanças...
